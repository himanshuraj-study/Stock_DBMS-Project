# Stock Management System (DBMS Project)

## Description

This project is a database-driven Stock Management System designed to manage and track inventory efficiently. It allows users to store, update, and retrieve product details such as stock quantity, price, supplier information, and transaction history.

## Features

* Add new products
* Update stock details
* Delete records
* View inventory
* Track stock inflow and outflow

## Technologies Used

* DBMS (MySQL/Oracle)
* SQL
* Java/Python (if used)

## Objective

To replace manual stock management with a digital system ensuring accuracy, efficiency, and data consistency.

## Conclusion

This system provides a simple and effective solution for inventory management and can be extended with advanced features like authentication and reporting.
